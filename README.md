# @Gridbolbot — Bollinger Grid Signals

Telegram-бот для торговых сигналов по стратегии Bollinger Grid.

**Возможности:**
- ⚡ Скальпинг x10 — быстрые входы по BB-полосам
- 🔄 Mean Revert — возврат к среднему
- 💰 Фандинг — мониторинг ставок финансирования
- 📊 Сканирование рынка по 200+ тикерам
- 💎 Pro-подписка через Telegram Stars + TON

**Бот:** [@Gridbolbot](https://t.me/Gridbolbot)

**Технологии:** Python, Bybit API v5, Bollinger Bands, Telegram Bot API
